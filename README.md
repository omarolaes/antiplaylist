## ⚡️ Quick Start

```bash
git clone https://github.com/omarolaes/antiplaylist.git
cd antiplaylist
npm install
npm run dev
```

## 🎯 Core Concept

Antiplaylist is the antidote to curated playlists. 

## 🛠 Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI & YouTube API
